2 · Decomposition method

Each layer is a closed outline the height of one sheet of MDF; stack and glue them in order.

3 · Real-world scale

Slices run along the vertical (Z) axis. Use the rotate buttons if your model is on its side — no need to go back to Tinkercad.

4 · Cutting parameters

Tabs stick up from one layer into a matching slot in the layer above — locks the stack against lateral sliding. Only applies to stacked-layers mode.
Kerf offset shrinks each outline inward slightly to compensate for blade width. Leave at 0 if unsure — hand-cut tolerance usually dominates anyway.
Each hex is a small interior cut-out — drill a starter hole, then jigsaw the hex outline. Skip if you need solid layers for strength.

5 · Print layout

Print at 100% for the model's real-world size. Use 50% for a miniature mockup or 200% to enlarge. The assembly preview and stats update to match.

6 · What to print

Useful for reprinting a single misprinted sheet, or splitting a big job across print sessions. Leave the full range to print everything.
⚠ 0 layer(s) exceed the selected paper size. They will be tiled across multiple pages.
